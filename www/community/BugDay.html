Bug Day

On Tuesday March 8th, 2005 from 8:00am to 4:00pm PST (03-08T16:00 - 03-09T00:00 UTC) the JXTA team will be participating in an online chat devoted to the upcoming release. If you are having problems with the new release, have questions for the developers or just want to stop by and see what's happening please stop by! Connect via the jxta-c discuss list or myJXTA2.

Or easier yet, just run it using java web start.
