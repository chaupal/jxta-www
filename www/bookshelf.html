Early Adopter JXTA, a book that teaches experienced developers all they need to know to be able to play confidently with JXTA. It also shows how to use JXTA as the foundation for developing a peer-to-peer application. By Sing Li. Available now from amazon.com.


JXTA: Java P2P Programming provides an invaluable introduction to this new technology at a time when developers crave useful information and practical examples. Created by members of the JXTA community, sharing their real-world experience, this book explains the fundamentals of P2P and demonstrates how JXTA fulfills the P2P promise. By Daniel Brookshier, Darren Govoni, Navaneeth Krishnan and Juan Carlos Soto. Available now from amazon.com


   

JXTA, a technical book by Brendon J. Wilson. Developed with input from the JXTA Community, this book provides comprehensive detail on the JXTA platform, illustrating key concepts to the reader using example code built on the Java reference implementation of JXTA. The complete book is available for free from the author's web site, along with the Java source code for all the book's examples. The book is also available to order from Amazon.com.


Mastering JXTA Development, a tutorial that is a code-intensive guide for Java developers. It demonstrates the process required to piece the JXTA technologies together to build sophisticated, real-world, peer-to-peer applications. By Joe Gradecki. Available from Wiley Publishers.


Java P2P Unleashed provides a single source for Java developers who want to develop P2P systems. The book explains the benefits of each technology and shows how to fit the P2P "pieces" together - both in building new systems and integrating with existing ones.

The book starts with a discussion of the P2P architecture, referencing similarities with existing, familiar systems while previewing several types of P2P applications. It explains how to plan ahead for security, routing, performance and other issues when developing a P2P application. Each technology included in the book - JXTA, Jini, JavaSpaces, J2EE, Web services - is approached from a P2P perspective, focusing on implementation concerns Java developers will face while using them. The last section includes several large-scale examples of different P2P applications - managing content, building communities, integrating services, routing messages, and using intelligent agents to gather information. The final chapter looks ahead to future developments in Java P2P technologies.

By Robert Flenner, Michael Abbott, Toufic Boubez, Navaneeth Krishnan, Alan Moffet, Rajam Ramamurti, Bilal Siddiqui, and Frank Sommers. Available from http://www.samspublishing.com Type the book's ISBN (0672323990) in the search window.


JXTA in a Nutshell, A Desktop Quick Reference.

Written by key members of Sun Microsystem's Project JXTA, JXTA in a Nutshell is the definitive reference to the most solid platform yet for Peer-to-Peer distributed computing. "P2P" enables users with the same networking application to connect with each other and directly interact with other people computers anywhere on the network. JXTA is a giant step forward in the evolution of P2P.

O'Reilly's pioneering reference is the first and last word on this powerful distributed computing technology. The book delivers all the information you need to get started, including an overview of P2P distributed computing, an explanation of the JXTA Project's new platform, and ways that developers can become a part of the development effort. It introduces major concepts in a hands-on way by explaining them in context to the shell, and contains a complete reference to the JXTA application bindings. Also included is the full JXTA protocol specification. The book covers important topics such as security, and how the JXTA technology fits into the standard Java classes.

By Scott Oaks, Bernard Traversat, and Li Gong, available from O'Reilly's Online Catalog and Amazon.com.
