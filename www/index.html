JXTA(TM) Community Projects

You can use JXTA(TM) technology to create peer-to-peer (P2P) applications based on Java technology. JXTA technology is a set of open protocols that enable any connected device on the network, ranging from cell phones and wireless PDAs to PCs and servers, to communicate and collaborate in a P2P manner. JXTA peers create a virtual network where any peer can interact with other peers and resources directly, even when some of the peers and resources are behind firewalls and network address translations (NATs) or on different network transports.

Project JXTA is fortunate to have a strong supportive community. Come join in the discussions and development efforts. There is such a wide range of activities to join in on that there is sure to be something of interest to you!
