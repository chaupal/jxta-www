Topologi Pty. Ltd.
Product: Topologi Collaborative Markup Editor
Description:

A JXTA-enabled XML/SGML/HTML editor with unique tools to help the special problems that
publishing teams and individuals face when marking up documents: large documents, fast editing, convenient markup, whitespace correction, working with ill-formed documents, etc.

The editor features state-of-the art validation languages, including Schematron, RELAX NG (full and compact syntax), WXS, Examplotron and DTDs.

JXTA has been used to:
- promote collaboration: workers can exchange and annotate screenshots, send messages, and view each other's directory diaries
- simplify administration: plugins can be updated over the peer network
- connect to other JXTA-based topologi server products; a Relayer product is available for Web access.
System Requirements: - Windows XP, Windows 2000/Pro/Server/Adv. Server, Windows ME, Windows NT 4.0 Service Pack 5, or Windows 98; Linux with KDE 3.n or Gnome desktop managers; Mac® OS X 10.1.n with JRE 1.3.1
- RAM: Windows and Linux: 128 MB (minimum), 256 MB (recommended); Macintosh: 64 MB (minimum)
- Disk space: Windows and Linux: 50 MB; Macintosh: 30 MB
- JRE 1.3.1 (preferred) or later. Windows distribution comes with bundled JRE.
Availability & Pricing: Available now for Windows, Linux and Macintosh
