Product: EConExplorer
Description: EConExplorer is a browser-based web application that can run on any platform (Linux, Unix, and Windows) using any JSP/J2EE container, and can be accessed using any web browser. It offers a P2P content network to author, manage, and distribute dynamic content from many resources such as: Legacy Systems, databases, portals, application servers, mainframes, and file-sharing systems. Any information stored in these systems such as raw data, documents, and media files are securely made available to end-users resulting in the creation of a "virtual desktop".
System Requirements: Any JSP container/J2EE Compliant Application Server, Intel Pentium II at 233 MHz or higher (minimum), 128 MB (minimum), 256 recommended, 100 MB Hard Drive Space.
