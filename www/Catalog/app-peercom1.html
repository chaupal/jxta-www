Peercom Co., Ltd.
Product: My Personal Portal
  - “My DeskStation” and “My Home” for PCs
  - “My WebSpace” & “Shared WebSpace” for Web-based views
Description:

By using JXTA’s network independent functionality, “My Personal Portal” makes it easy to view digital content that is stored in many digital devices (home pc, office pc, mobile pc, etc.)
- Content Replication “My DeskStation” helps replicating content between peer PCs. “My WebSpace – for person” and “Shared WebSpace – for group” make possible to up/download content directly using "My DeskStation" or a web browser just like internet disk service does. Users can find content without any restriction of space. Content is available to the user regardless of its source.
- Variable Content Types Digital content is sorted in by the type (file, audio, video, photo album type), and personal PIMS data (my favorites, memo, address, mail) is also replicated by its type.
- Micro HomePage Servcie Allows users to access any PC and use any content in that PC anytime anywhere by using web browser.
- DCRL (Digital Content Resource Location) The concept of DCRL function provides a direct link service on personal PCs. Large files(video, audio...) stored in a PC can be serviced(accessed) by our uniformly content addressing schema (DCRL). DCRL can be linked or embeded in many web pages freely and the digital content referenced by DCRL can be accessed whenever the PC is online. For example, those large files can be delivered to a desired destination as a DCRL link attachment in e-mail.
System Requirements: - Windows XP, Windows 2000, Windows ME, Windows 98 SE with JRE 1.4.X
- RAM: Windows 128 MB (minimum), 256 MB (recommended)
- Disk space: Windows 50 MB
- JRE 1.4 (preferred) or later
Availability & Pricing: Available now for Windows PC
Peercom is planning to develop another version for PDA, Mobile phones. A demo (in Korean) is available.

Target Markets: - General users who use multiple pc devices
- Web disk (Internet disk, web hard) users
- Enterprise users in need of file sharing between team members
- Users who share large multi media files (audio, video)
