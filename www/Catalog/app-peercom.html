Product: My Personal Portal
  - “My DeskStation” and “My Home” for PCs
  - “My WebSpace” & “Shared WebSpace” for Web-based views
Description:

A By using JXTA’s network independent functionality, “My Personal Portal” makes it easy to view digital content that is stored in many digital devices (home pc, office pc, mobile pc, etc.)
- Content Replication “My DeskStation” helps replicate content between peer PCs. “My WebSpace – for person” and “Shared WebSpace – for group” make possible to up/download content using a web browser just like web disk service does. Users can find content without any restriction of space. Content is available to the user regardless of its source.
- Variable Content Types Digital content is sorted in by the type (file, audio, video, album type), and personal PIMS data (my favorites, memo, address, mail) is also replicated by its type.
- Micro HomePage Servcie Allows users to use PC content anytime anywhere by micro homepage function.
- DCRL (Digital Content Resource Location) The latest concept of DCRL function provides a direct service on personal PCs. Large files stored in PC can be serviced by creating a link that can be controlled from outside through static (without another server service.) For example, those large files can be delivered to a desired destination as an attachment in mail.

System Requirements: - Windows XP, Windows 2000, Windows ME, Windows 98 SE with JRE 1.4.X
- RAM: Windows 128 MB (minimum), 256 MB (recommended)
- Disk space: Windows 50 MB
- JRE 1.4 (preferred) or later
Availability & Pricing: Available now for Windows PC
Peercom is planning to develop another version for PDA, Mobile phones. A demo (in Korean) is available.

Target Markets: - General users who use multiple pc devices
- Web disk (Internet disk, web hard) users
- Enterprise users in need of file sharing between team members
- Users who share large multi media files (audio, video)
