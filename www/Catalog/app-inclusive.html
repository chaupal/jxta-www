Product: Momentum[tm]
Description: Momentum is the planning and tracking tool for the electronics industry and is used throughout the entire product lifecycle. System Requirements: - Operating System: Windows XP, Windows 2000, Windows ME, Windows NT 4.0 Service Pack 6, or Windows 98 SE
- RAM: 128 MBs (minimum); 256 MBs (recommended)
- Network Card: 10 MIPS (minimum); 10/100 MIPS (recommended)
Availability & Pricing: A pre-release program is currently underway; the commercial release is scheduled for Summer, 2002.
Target Markets: Electronics industry
