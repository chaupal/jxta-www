Improv Technologies
Product: Cirquet[tm] Solution Suite
Description: The Cirquet Solution Suite is a breakthrough set of infrastructure products and services that enables dynamic businesses to rapidly and cost effectively develop, deploy and manage scalable distributed applications.

Cirquet technology enables software developers to easily create Web service-based distributed applications from any Java[tm] language component. It wraps each Java component as a discrete application and allows for their assembly at run-time. Applications written using Cirquet Technology are network and device neutral; significantly increasing deployment flexibility and code re-use. The Cirquet Solution Suite utilizes JXTA technology as the distributed message transport to service n-Tier and P2P architectures. In addition, Cirquet updates, upgrades or bug fixes can be centrally managed and automatically propagated across the network to any device running the component without the need to shut down the application, computer or network.
System Requirements: Windows NT, 2000, XP, 98, Solaris 8, Linux 7.2
