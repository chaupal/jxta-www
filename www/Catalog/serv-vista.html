VistaPortal Software, Inc.
Product: VistaRepository
Description: VistaRepository provides a unified and secure environment in which various services can operate to provide distributed or centralized storage and scalable, flexible access to objects of different kinds including file systems, LDAP and relational database management systems (RDBMS), as well as spreadsheet data (StacCalc and Excel). VistaRepository facilitates peer-to-peer communication, file sharing, and network collaboration by utilizing JXTA based Vista Repository Server to create distributed working area where users can access, write and manipulate data from files and databases stored on remote serves or peer workstations. VistaRepository is part of the Vista Reporting Solution offered by VistaPortal Software, Inc.
